# COS 598I: Modern Data Systems (Spring 2026)

**Classes:** Mondays/Wednesdays 2:55pm – 4:15pm in Jones Hall 100

**Instructor:** [Jialin Ding](https://jialinding.github.io/)

**Office hours:** Directly after class or by appointment (email Jialin)

**Quick links**: [syllabus](https://docs.google.com/document/d/19qY10QB8LeqYXuF6RScIIJUgw6XFbLai1J-f5iFm9es/edit?usp=sharing) / [schedule](https://docs.google.com/spreadsheets/d/1yuwLoTLSAKx-FwCD4fNYRmly6-6Ah3XCVhkbGyn0xNg/edit?usp=sharing) / [Ed](https://edstem.org/us/courses/94107/discussion) / [discussion questions form](https://forms.gle/wkiLn3zmz9CacPpr9)

**Description**: Data systems are booming, and it's so much more than the SQL and relational databases that you learned in your undergraduate Intro to Databases class. This course provides a sampling of big ideas in modern data systems, including large-scale analytic data systems, data lakes and lakehouses, cloud-native data systems, NewSQL/HTAP/in-memory databases, query execution optimizations, and recent trends in using AI to improve data systems and building data systems that support AI applications.

Students are expected to participate in interactive paper discussions and to complete a course project in a topic related to data systems.
